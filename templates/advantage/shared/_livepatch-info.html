Ubuntu 14.04 ESM and 20.04 LTS

To activate Livepatch: ua enable livepatch
To deactivate: ua disable livepatch

Ubuntu 16.04 LTS and 18.04 LTS

To activate Livepatch: sudo snap install canonical-livepatch && sudo canonical-livepatch enable {{ contract['token'] }}
To deactivate: sudo canonical-livepatch disable

Other Ubuntu versions

Livepatch is available only on LTS versions. Consider upgrading to Ubuntu 20.04 LTS.
